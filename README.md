# Sintra Travel Guide 🏰

A beautiful, responsive website for exploring Sintra, Portugal's magical UNESCO World Heritage town.

## 🌐 Live Website

**Visit:** [https://stevenhuang33.github.io/sintra-guide/](https://stevenhuang33.github.io/sintra-guide/)

## ✨ Features

- **Responsive Design** — Works perfectly on desktop, tablet, and mobile
- **6 Major Attractions** — Detailed information on Pena Palace, Quinta da Regaleira, Moorish Castle, and more
- **One-Day Itinerary** — Optimized schedule to make the most of your visit
- **Practical Information** — Transportation, tickets, restaurants, and timing tips
- **Insider Tips** — Essential advice for first-time visitors
- **Modern UI** — Smooth animations, gradient colors, and interactive cards

## 🏛️ Attractions Covered

1. **Pena Palace** — Iconic colorful Romanticist palace
2. **Quinta da Regaleira** — Gothic estate with mysterious Initiation Well
3. **Moorish Castle** — Medieval fortress with panoramic views
4. **Sintra National Palace** — Portuguese Renaissance palace with twin chimneys
5. **Monserrate Palace** — Exotic palace with botanical gardens
6. **Cabo da Roca** — Westernmost point of mainland Europe

## 🚀 Quick Start

Simply open `index.html` in any modern web browser, or visit the live site above.
